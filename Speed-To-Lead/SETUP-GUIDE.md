# Butter Code Speed-To-Lead™ — Setup Guide (V2)
### Free-stack build: Google Forms + Google Sheets + Gmail + Make.com

**Product**: Butter Code Speed-To-Lead™
**Price**: R2,500–R4,000/month
**Promise**: Respond to every lead in under 60 seconds. Follow up automatically for 30 days. Recover closed leads at 90 days.

---

## What's included

- **Scenario 1 — Lead Capture**: instant trigger on Google Form submission. 3-tier lead scoring (Hot/Warm/Cold), auto-reply to the lead, and immediate owner alert.
- **Scenario 2 — Follow-up Engine**: runs once daily. 5-stage email sequence — Day 1, 3, 7, 14, 30. Marks lead CLOSED after Day 30 with no response.
- **Scenario 3 — Lost Lead Recovery**: runs quarterly (or manually). Re-engages leads that went CLOSED 60–90 days ago. Restarts the nurture sequence on response.
- **Leads sheet template**: 15-column Google Sheet that acts as a lightweight CRM.

---

## Lead Scoring — 3 Tiers

| Tier | Score | Keywords (in enquiry message) |
|---|---|---|
| Hot | 9 | urgent, today, asap, now, emergency, book |
| Warm | 6 | price, quote, interested, looking, services, info, available |
| Cold | 3 | Everything else |

Customise keywords per client. A solar company might add "loadshedding", "panels" to Hot. A salon might add "appointment", "this week".

## Setup steps (do this once per client)

### 1. Create the Google Form
Create a new Google Form with these exact field names (must match exactly, including capitalisation):

- **Full Name** (short answer)
- **Email** (short answer, set validation to "Email")
- **Phone** (short answer)
- **Service Needed** (short answer or dropdown)
- **Message** (paragraph)

Get the Form ID from the URL (long string between `/d/` and `/edit`).

### 2. Create the Leads Google Sheet
Create a new Google Sheet. Add a tab named **Leads**. Paste the header row from `leads-sheet-template.csv`:

```
Lead ID, Timestamp, Full Name, Email, Phone, Service Interest, Message, Lead Score, Hot Lead, Lead Tier, Status, Last Contacted, Next Follow-up, Follow-up Count, Notes
```

Get the Spreadsheet ID from the URL.

### 3. Import Scenario 1 (Lead Capture)
1. Create a new scenario in Make → Import Blueprint → upload `scenario-1-lead-capture.json`
2. Replace `REPLACE_WITH_FORM_ID` with the Form ID from step 1
3. Replace `REPLACE_WITH_SPREADSHEET_ID` with the Spreadsheet ID from step 2
4. Replace: `[BUSINESS NAME]`, `[OWNER NAME]`, `[OWNER EMAIL ADDRESS]`, `[PHONE NUMBER / WHATSAPP NUMBER]`
5. Connect the client's Google account (Forms + Sheets) and Gmail account
6. Activate the scenario

### 4. Import Scenario 2 (Follow-up Engine)
1. Import `scenario-2-follow-up-engine.json`
2. Replace `REPLACE_WITH_SPREADSHEET_ID` and all name/email placeholders
3. Set the schedule to run **once daily** (e.g. 09:00, client's local timezone)
4. Activate

### 5. Import Scenario 3 (Lost Lead Recovery)
1. Import `scenario-3-lost-lead-recovery.json`
2. Replace `REPLACE_WITH_SPREADSHEET_ID` and all name/email placeholders
3. Set the schedule to every **90 days** (or run manually once per quarter — uses minimal ops either way)
4. Activate

### 6. Test it
1. Submit a test form entry using a Hot keyword in the message (e.g. "I need this today")
2. Confirm: instant reply email arrives, owner gets a "🔥 HOT LEAD" alert, a row appears in Leads with Lead Tier = Hot
3. Submit another entry without keywords — confirm Lead Tier = Cold, score = 3
4. Set a test row's "Next Follow-up" to today, run Scenario 2 manually — confirm Day 1 email sends and count increments to 1
5. Set count to 4 and run again — confirm Day 30 email sends and Status = CLOSED

## Handover checklist (30-minute client walkthrough)

- [ ] Show the client their Google Form and how to share/embed the link (website, Facebook, WhatsApp Business)
- [ ] Show the Leads sheet — explain Lead Tier, Status, and Hot Lead columns
- [ ] Closing a deal: owner sets **Status = WON** in the sheet → stops all follow-ups
- [ ] Scenario 3 (Lost Lead Recovery) runs quarterly automatically and re-engages closed leads — no action needed from the owner
- [ ] Show where to edit email wording inside Make if the client wants to tweak tone later

## KPI tracking (what to review monthly)

| Metric | How to measure |
|---|---|
| Lead response time | Check timestamps in Leads sheet — Timestamp column vs. time owner first replied |
| Lead Tier distribution | Count rows by Lead Tier column — track Hot % over time |
| Conversion rate | Count WON / total leads |
| Follow-up drop-off | Count leads at each Follow-up Count stage |
| Lost lead recovery | Count RE-ENGAGED leads generated by Scenario 3 |
